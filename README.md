# DataCo Sales & Customer Analysis (Excel Project)

## Project Overview

This project simulates the role of a data analyst working with DataCo datasets. The goal was to clean, transform, and analyze sales, customer, and product data using Microsoft Excel, and to build visual dashboards for business insights.

---

## Objectives

* Prepare and clean raw datasets (customers, products, orders)
* Perform data transformation using Excel functions
* Analyze sales performance and customer behavior
* Build dashboards and visualizations to support decision-making

---

## Tools & Skills Used

* Microsoft Excel
* Data Cleaning & Preparation
* Pivot Tables & Pivot Charts
* Excel Functions (IF, AND, NOT, LOOKUP)
* Data Visualization (Bar, Line, Doughnut, Scatter, Waterfall charts)
* Dashboard Design

---

## Data Preparation

### Cleaning & Structuring Data

* Imported datasets (Customers, Products)
* Removed duplicates using unique IDs
* Standardized formats (dates, currency, percentages)
* Created a **Full Name** column and removed redundant fields

### Data Transformation

* Combined address fields into a single column
* Created calculated columns such as:

  * Scheduled Delivery Date
  * Customer Address

---

## Data Analysis Using Functions

* Identified **expensive products** using conditional logic
* Created flags for:

  * On-time priority deliveries
  * High-value cash transactions (> $200)
* Corrected revenue calculations by incorporating discounts
* Computed total sales using derived metrics

---

## Data Enrichment & Exploration

* Used lookup functions to:

  * Add Product Name
  * Add Product Category
* Built PivotTables for exploratory data analysis

---

## Data Visualization

### Basic Charts

* Regional sales vs profit comparison
* Sales by product sub-category (Top: Phones, Chairs, Storage)
* Sales distribution by customer segment
* Quarterly regional performance

### Advanced Charts

* Combo charts (column + line)
* Waterfall charts

## Dashboard Development

## Key Insights

* Central region generates high sales but lower profit contribution
* A small number of product categories drive most revenue
* Certain transactions (cash + high value) indicate key customer segments
* Discounts significantly impact actual sales performance
